# Infrastructure as a Service

IaaS is an on-demand, pay-as-you-go (Often usage-based, or based on cycle usage) service that offers:

- compute,
- storage,
- networking resources

# IaaS Providers

- DigitalOcean
- AWS
- Linode
- Synology
- Alibaba
- Azure
- Google Cloud
- IBM Cloud
